Configuration de la carte :

Ouvrez cette page avec votre éditeur html ou avec le bloc-notes :

Chaque régions est délimitée par les lignes <area shape="... >

Dans chacune de ces lignes, vous pouvez modifier :

  • le lien (url) href="http://www.google.fr?q=région picardie"
  • le mode d'ouverture du lien : target="_self"
  • le titre : title="Picardie"

Cette carte convient aux sites statiques ou dynamiques.
